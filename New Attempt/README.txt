Run the code in a specific sequence.


After extracting the zipped folder, run "CannyEdgeDetector.py" file. You will be prompted to enter the filename. Make sure that the image file is in the same location as that of the code. Also width and height of the image must be equal. You will see several output images as the result. Result from each step of the Canny edge detector is saved in the location.
The final result of this code, is a file named as "EdgeImage.jpg".

Now run the "HoughSpace.py" file and the correspoding hough space you will get for an image.

To detect the number of rectangles in the given image, a separate code named as "RectangleDetection.py" is written. Run this code to see the output.
